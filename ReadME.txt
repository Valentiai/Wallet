Добро пожаловать в приложение Wallet.
Приложение Wallet предназначено для ведения домашней бухгалтерии.

В качестве хранилища записей используется файл Database.txt.
Файл автоматически создается и хранится в той же директории, где находится исполняемый файл main.py.
В случае перезапуска приложения оно автоматически загружает данные из файла Database.txt без создавания нового.
Вместо этого, приложение работает с уже существующей БД.

При запуске приложения оно приветствует Пользователя, кратко описывает свой функционал.
Далее Пользователь видит текущее состояние баланса, доходы и расходы.

Список и описание доступных функций:.
1) Запрос баланса
Для вывода баланса пользователю необходимо ввести команду balance.
Приложение выводит текущее состояние баланса, доходы и расходы.
2) Добавление в БД новой записи
Для добавления в БД записи о приходе или расходе средств Пользователю необходимо ввести команду add.
Пользователю предлагается ввести:
- Сумму операции
- Категорию операции (доход или расход)
- Описание операции (текстовая заметка в свободной форме)
После успешного добавления операции в БД Пользователю выводится актуальное состояние счета.
Пользователь возвращается в главное меню.
3) Поиск по записям
Для поиска записи среди существующих Пользователю необходимо ввести команду search.
Для поиска записи Пользователю предлагается ввести ключ поиска:
- ID записи (16-значное число)
- Категория записи (доход или расход)
- Дата записи (вводится в формате ГГГГ-ММ-ДД)
- Сумма операции. Для поиска по сумме возможны 2 опции: поиск по точной сумме, поиск суммы в заданном диапазоне
Программа выводит найденные записи в формате словаря. Каждый элемент записи с новой строки.
4) Редактирование или удаление записи.
Для редактирования или удаления сущесмтвующей записи Пользователю необходимо ввести команду edit.
Для дальнейшей работы Пользователю предлагается ввести 16-значный ID записи.
Если ID не известен, Пользователю предлагается воспользоваться функцией Поиск по записям.
Если ID найден в базе, предлагаются 2 опции:
- удалить
Удаление происходит без запроса дополнительного подтверждения.
- изменить
Для изменения записи Пользователю предлагается ввести:
- новую сумму записи
- новую категорию записи (доход или расход)
- новое описание записи (комментарий в свободной форме)
Новая запись помещается в конец списка записей. Ей присваивается прежний ID.
После вышеуказанных манипуляций (удаление/изменение) в БД также обновляются:
- баланс
- доходы
- расходы
- количество имеющихся записей
5) Удаление всех записей
Для очистки БД Пользователю необходимо ввести команду clear.
Перед очисткой Пользователь должен ввести для дополнительного подтверждения:
- Y
Происходит удаление существующей БД и создание новой.
- N
Пользователь возвращается в главное меню.
6) Главное меню
Для возврата в главное меню Пользователю необходимо ввести команду menu.
7) Выход
Для завершения работы приложения Пользователю необходимо ввести команду exit.
